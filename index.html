由于浏览器安全限制，本地生成的二维码无法在移动设备上直接扫描打开。请将本系统部署到服务器（如GitHub Pages、Vercel等）后使用。

GitHub Pages部署

  1. 登录GitHub账户，创建一个新仓库
  2. 将本HTML文件上传到仓库
  3. 在仓库设置中启用GitHub Pages功能
  4. 通过生成的GitHub Pages链接访问系统
GitHub Pages指南

Vercel部署

  1. 访问 vercel.com 并注册
  2. 导入您的GitHub仓库
  3. 选择项目并点击部署
  4. 通过Vercel提供的域名访问系统
Vercel文档
注意：部署后请确保使用HTTPS链接访问系统，移动设备才能正常扫描二维码。
