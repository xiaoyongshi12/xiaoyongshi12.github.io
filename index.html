🚀 部署指南

1. 保存HTML文件

将本页保存为HTML文件（例如：info-system.html）

2. 选择部署方式

选择一种部署方式：

  • GitHub Pages：免费静态网站托管服务
  • Netlify/Vercel：专业的静态网站托管平台
  • 自有服务器：使用Nginx/Apache部署

3. GitHub Pages部署步骤

a. 创建GitHub账户（如果没有）
b. 创建新仓库
c. 上传HTML文件到仓库
d. 进入仓库设置 → Pages → 选择main分支 → 保存

4. 访问部署的网站

部署完成后，访问类似 https://username.github.io/repository/ 的URL

5. 使用二维码功能

在部署后的网站上生成二维码，即可通过移动设备扫描访问
