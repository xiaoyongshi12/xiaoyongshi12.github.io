GitHub Pages部署指南

1. 创建新仓库

登录GitHub账户，点击右上角的"+"图标，选择"New repository"。

在创建仓库时，选择以下设置：

  • 仓库名称：输入一个有意义的名称（如info-system）
  • 选择"Public"（公开）
  • 勾选"Add a README file"
  • 点击"Create repository"按钮

2. 上传HTML文件

在您新创建的仓库页面：

  1. 点击"Add file"按钮，选择"Upload files"
  2. 将您的HTML文件（可命名为index.html）拖拽到上传区域
  3. 添加一个简短的提交信息，例如"Add info system"
  4. 点击"Commit changes"按钮

3. 配置GitHub Pages

在仓库页面：

  1. 点击顶部的"Settings"选项卡
  2. 在左侧菜单中找到"Pages"选项
  3. 在"Source"部分，选择"Deploy from a branch"
  4. 在"Branch"下拉菜单中，选择"main"分支和"/ (root)"文件夹
  5. 点击"Save"按钮

4. 访问您的网站

配置完成后：

  • 返回"Settings" → "Pages"页面
  • 等待看到"Your site is published at..."消息
  • 点击提供的链接访问您的网站

您的网站URL将类似于：

https://<your-username>.github.io/<repository-name>/
